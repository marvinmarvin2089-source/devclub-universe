## ✨ Sobre o Projeto

O DevClub Universe é uma experiência interativa criada para mostrar que aprender programação é como explorar um universo inteiro de possibilidades.

Mais do que uma página institucional, o projeto utiliza storytelling, animações, efeitos visuais e uma narrativa imersiva para transmitir a essência do DevClub:

> Transformar vidas através da tecnologia.

- Logo interativa.

- Área do aluno.
- Formações.
- Responsividade completa.

## 💻 Rodando Localmente

```bash

git clone https://github.com/marvinmarvin2089-source/devclub-universe.git

# Entrar na pasta
cd devclub-universe

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```
